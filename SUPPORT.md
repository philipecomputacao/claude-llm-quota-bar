# Support

Need help with `claude-llm-quota-bar`? Here's where to ask:

## Questions & general help

**GitHub Discussions tab** — open a thread for usage questions, ideas, or
"how do I…?" topics. The maintainer reads everything; community answers
often land first.

## Bug reports

**GitHub Issues → "🐛 Bug report" template** — please include the bar
output (paste the lines verbatim), your platform, and the provider
you were testing. The full checklist is in the template.

## Feature requests

**GitHub Issues → "💡 Feature request" template** — describe what
you're trying to do, why the current setup makes it hard, and (if you
can) sketch the desired bar layout in ASCII.

## Security disclosures

**Do not open a public issue.** Email the maintainer privately using
the address in the GitHub profile. See [SECURITY.md](SECURITY.md) for
the full policy and a recent audit log.

## Out of scope

The following are **not** support channels for this project:

- The `free-claude-code-minimax` repo (a different project — see the
  [README § Related projects](README.md#related-projects) link)
- The upstream `Miluer-tcq/cc-statusline` repo (this project forked from
  it but is now independent — see [README § Upstream history](README.md#upstream-history))
